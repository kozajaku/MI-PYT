twitterjk tool documentation
============================

Welcome to the documentation of tool **twitterjk**. This tool was created for the purposes of MI-PYT lecture at FIT CTU
college.

The tool is simple program written in Python that is able to query tweets through Twitter API and show them
in readable format to the user in console or through simple web GUI.

Sources can be found in the repository
`kozajaku/MI-PYT`__ on the GitHub. Also the packaged Python module can be downloaded from testpypi_

.. _repo: https://github.com/kozajaku/MI-PYT
__ repo_
.. _testpypi: https://testpypi.python.org/pypi/twitterjk
